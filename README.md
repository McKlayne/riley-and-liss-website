# Riley and Liss - Seeking a Gestational Carrier

A Next.js website to help Riley and Liss find a compassionate gestational carrier to help complete their family.

## About This Project

This website serves as an introduction to Riley and Liss, a married couple of 7 years who are seeking a gestational carrier due to medical reasons that prevent Liss from carrying a child. The site provides:

- Personal information about Riley and Liss
- Photo gallery showcasing their life together
- Information about the gestational carrier process
- Requirements and steps involved
- Contact information
- Link to their GoFundMe campaign

## Key Features

- **Responsive Design**: Works on all devices
- **Photo Gallery**: Interactive carousel showcasing their life
- **Clear Information**: Detailed process and requirements
- **Contact Integration**: Direct phone number links
- **Professional Presentation**: Clean, warm design appropriate for the sensitive nature of the request

## Technical Details

- **Framework**: Next.js 16 with React 19
- **Styling**: Tailwind CSS with custom color scheme
- **UI Components**: Radix UI components with shadcn/ui
- **Analytics**: Vercel Analytics integration
- **Deployment**: Optimized for Vercel hosting

## Getting Started

```bash
# Install dependencies
npm install
# or
pnpm install

# Run development server
npm run dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to view the website.
